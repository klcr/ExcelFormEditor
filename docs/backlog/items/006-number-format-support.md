# 006-数値フォーマット対応

## 状態
完了

## 担当
（未定）

## 対象パッケージ
web / domain

## 概要

XLSX の数値フォーマット（numFmt）をパースし、日付・通貨・パーセントの表示形式をセル値に反映する。

## 背景

現在のパーサーはセル値を `<v>` 要素の生値で取得する。日付セルは Excel シリアル値（例: `45000`）がそのまま表示され、パーセントは小数（例: `0.15`）で表示される。帳票にこれらのフォーマットが含まれる場合、ユーザーが手動で値を修正する必要がある。

## 前提条件チェックリスト
- [x] 軽量 XLSX パーサーが動作していること
- [ ] テーマカラー対応（005）が完了していること（必須ではないが推奨）

## 実装チェックリスト
- [ ] `StylesParser.ts`: `<numFmts>` テーブルをパース（`numFmtId → formatCode` マップ）
- [ ] 組み込みフォーマット ID（0-49）のマッピングテーブルをハードコード
- [ ] 日付判定: formatCode に `y`/`m`/`d`/`h`/`s` を含むかで簡易判定
- [ ] Excel シリアル値→日付変換: `(serial - 25569) * 86400000`（1900/2/29 バグ考慮）
- [ ] `CellValueResolver.ts`: numFmtId に基づくフォーマット適用
- [ ] `ExcelTypes.ts`: `RawCell` に `numFmtId` を追加
- [ ] テスト作成

## 備考

完全なフォーマット再現は目指さない。日付・パーセント・通貨の 3 パターンをカバーすれば 80% 対応。

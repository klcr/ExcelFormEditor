# 002-座標原点 — 印刷可能領域基準

## 発生日
2026-03-05

## 対象レイヤー
domain

## 概要
ボックス座標の原点は用紙物理端ではなく、余白を除いた印刷可能領域の左上 (0, 0) とする。

## 背景
Excelのセル配置は印刷可能領域（余白の内側）を基準としている。座標原点を印刷可能領域に合わせることで、Excel上のセル位置とボックス座標が自然に対応する。また、余白設定を変更してもボックス座標自体の再計算が不要になり、出力時のオフセット加算のみで済む。

## 詳細

### 座標系の定義

```
用紙物理範囲（例: A4 portrait = 210×297mm）
┌─────────────────────────────────────┐
│         margin_top                  │
│  ┌─(0,0)──────────────────────┐     │
│  │ 印刷可能領域                │     │
│m │                            │m    │
│l │  box(x=5, y=10, w=50, h=8)│r    │
│  │                            │     │
│  └────────────────────────────┘     │
│         margin_bottom               │
└─────────────────────────────────────┘
```

- ボックスの `x`, `y` は印刷可能領域の左上からの距離（mm）
- 印刷可能領域のサイズ:
  - `printableWidth = paperWidth - (marginLeft + marginRight) × 25.4`
  - `printableHeight = paperHeight - (marginTop + marginBottom) × 25.4`

### CSS 出力時の変換

```
CSS left = marginLeft(mm) + x
CSS top  = marginTop(mm) + y
```

### エディタ上の表示

- キャンバスは印刷可能領域のサイズで描画する
- 余白領域はグレーアウトして表示する（将来の UI 拡張）

### 制約 001 との関係

- 001（mm絶対座標）が座標の単位を定義
- 本制約（002）が座標の原点を定義
- 両方を組み合わせることで「印刷可能領域の左上を原点とする mm 絶対座標系」が確定する

## 影響範囲
- `src/domain/paper/` — 印刷可能領域の算出、座標変換
- `src/domain/box/` — ボックス座標の解釈（印刷可能領域基準）
- `src/domain/line/` — 線分座標の解釈（印刷可能領域基準）
- `src/web/` — エディタキャンバスの描画範囲、CSS 出力時のオフセット加算

## 今後の対応
設計の根幹であり、解消予定なし。変更する場合は制約 001 と同様に全レイヤーへの影響を評価する必要がある。
